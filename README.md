# Genshin Data API

## Description

As Spring practice, implementing an API that serves the data needed to mimic Project Amber.

## Supported

* Characters

## Notes

Latest version of Genshin tested: 5.4, upgrading to 5.5 is not possible due to the new obfuscation methods.
